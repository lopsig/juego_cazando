El Juego del Gato

Autor: LopSig Dev

Instrucciones:

- Pulsa el botón "Iniciar Juego" para comenzar
- Caza 6 veces al ratón para ganar el juego
- Pulsa los botones ⬆, ⬇, ⬅, ➡ para mover al gato
- Pierdes si se acaba el tiempo y el juego se reinicia automaticamente
- Puedes reiniciar el juego cuando quieras
- Pulsa "Detener" para acabar el juego
